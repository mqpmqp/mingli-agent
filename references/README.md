# References

本目录保存原始来源，不保存运行时规则或已整理知识。

- `books/`：原始书籍和出版物。
- `papers/`：论文、研究报告和可引用文章。
- `courses/`：课程资料及其原始压缩档案。
- `screenshots/`：来源明确的原始截图。
- `images/`：扫描图和其他原始图片。
- `cases/`：经许可、去敏后的原始案例资料。

导入前必须先运行资料 inventory，检查许可、隐私、SHA-256、重复文件和体积。PDF、图片和压缩档案使用 Git LFS；提交前须安装并运行 `git lfs install`。

不要修改原始来源，不要把未经整理的 PDF/OCR 转换结果直接当作规则真值。结构化整理结果应进入 `knowledge/`，并保留可回溯的来源标识。
