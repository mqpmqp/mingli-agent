# Ziwei Traditional Rule Content v1 来源与转述边界

## 登记来源

规则资源统一引用 `classical:ziwei-doushu-quanji-star-palace`，登记名称为《紫微斗数全书》星曜与十二宫传统义理索引。该 source ID 表示传统义理来源类别，不表示每条短句是古籍逐字引文，也不表示传统内容经过现代实证验证。

## 转述方法

- 十四主星×十二宫共 168 个 pair 逐格编写原创短句，不复制长段原文。
- 每条短句把传统术语转换成可观察的行为倾向、边界和现实核验建议。
- 四化、亮度与同宫组合只提供最小版本化修饰，不扩张为完整格局或事件预测。
- 内容生成脚本保存显式 14×12 矩阵；跟踪 JSON 必须与脚本输出完全一致。
- 生命周期固定从 `draft` 开始，不能因测试通过自动升级为 `reviewed` 或 verified。

## 禁止边界

- 不复制 Metis 或其他第三方平台的排盘、解释或界面输出。
- 不批量抓取、OCR 或拼接外部内容。
- 不保存长段古籍原文；仓库只保留 source metadata 和原创结构化短句。
- 不使用“一定、必然、注定、保证”等绝对语言。
- 不把传统规则用于医疗诊断、投资指令、法律决定或婚姻决定。
- 不把规则覆盖率解释为预测准确率；`prediction_validity` 固定为 `not_evaluated`。

## 审查状态

当前 184 条规则均为 `draft`。工程门禁只证明 Schema、来源引用、行为匹配、冲突处理、打包和防伪覆盖可复现；Rule Content Hold 在独立内容审查与 PR B 合并前保持 ACTIVE。
