# Productization PRD

## 产品目标

提供可追踪、现实优先、非宿命化的传统命理分析工具。

## MVP 用户流程

1. 用户创建命盘档案
2. 输入出生资料并选择是否校正真太阳时
3. 系统生成确定性盘面
4. 用户确认四柱与日主
5. 选择主题：综合、事业、财运、感情、考公、创业、迁移
6. 补充现实信息
7. 系统输出带置信度的分析
8. 用户可反馈已发生事件，进入匿名案例审核队列

## MVP 页面

- 首页
- 新建命盘
- 命盘确认
- 主题问答
- 分析报告
- 历史报告
- 反馈与纠错
- 管理后台：规则、案例、知识版本

## 不进入 MVP

- 付费法事或改运商品
- 彩票、赌博和具体投资买卖建议
- 死亡、灾祸或疾病预测
- 未经验证的自动学习直接上线
- 自动抓取私人资料

## 成功指标

- 四柱确认率
- 报告完成率
- 现实信息填写率
- Benchmark 分数
- 用户纠错率
- 绝对化表达违规率
