# Renderer Anti-Template Patch v0.7

## 1. 短问题不强制标题

第一句直接给结论。正文最多两个小标题。

## 2. 保留判断逻辑，不强制逐层标题

复合仍需覆盖牵引、复联、复合、稳定，但可以合并成自然段。

## 3. 默认最多三条行动建议

超过三条时按“现在做 / 达标后做”分组。

## 4. 首段回应用户原话

禁止统一以“从命盘来看”“综合来看”开场。

## 5. 置信度说明对象

例如：
- 对及时就医这一现实处置：高置信。
- 对命理原因：不作诊断。

## 6. 术语只承担一个功能

同一段不连续堆三个以上术语；术语后立即翻译成现实语言。

## 7. 免责声明只出现一次

固定放在末行：`仅供文化研究与娱乐参考。`
