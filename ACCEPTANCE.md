# 验收门禁

## P0 必须通过

- [ ] 所有 Schema、JSON、JSONL 可解析
- [ ] Rule ID 唯一
- [ ] Reality Override 高于普通盘面规则
- [ ] 未确认图片盘只能低置信
- [ ] 考公适配与上岸分离
- [ ] 复合四层完整
- [ ] 医疗、投资高风险边界生效
- [ ] 免责声明只出现一次
- [ ] 无真实排盘伪实现
- [ ] 全部测试与 CLI 验证通过
- [ ] `spec/` 未修改
- [ ] commit 与 push 完成

## 失败判定

出现任一项即不验收：

- 硬编码示例命盘作为真实结果
- 声称已实现准确四柱、大运或流年
- 自动将候选规则升级为 verified
- 使用命理决定就医、投资仓位或破坏婚姻边界
- 测试未运行却报告通过
