# Knowledge

本目录只保存经过整理、可检索并能回溯来源的 Markdown 或结构化数据。

领域目录：

- `yuan/`：渊源体系整理。
- `bazi/`：八字结构化知识。
- `ziwei/`：紫微结构化知识。
- `qimen/`：奇门结构化知识。
- `fengshui/`：风水结构化知识。
- `yijing/`：易经结构化知识。
- `rules/`：经过规则生命周期管理的候选或已审核规则。
- `benchmark_notes/`：基准案例的人工观察与复核记录。

每项资料应记录来源 ID、具体页码或片段、整理方式和审核状态。未经复核的 OCR、PDF 转换文本或模型摘要不得当作规则真值，也不得自动标记为 `verified`。

原始 PDF、扫描件、截图、图片和压缩档案应放在 `references/`，不能放入本目录。
